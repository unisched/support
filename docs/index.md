# Головна сторінка

**UniSched** — це зручний мобільний та веб-застосунок для перегладу вашого розкладу занять, створений для студентів та викладачів закладів вищої та професійно-технічної освіти України. Його покликанням є надати студентству та освітянам зручний спосіб переглядати свої поточні заняття, знаходити своїх викладачів та колег серед десятків, а то і сотень кабінетів тоді, коли це необхідно — всього у два кліки в застосунку UniSched.
