# Для закладів освіти

Наша команда [Громадської організації «Верховний Порядок»](https://supremeorder.rocks/) запрошує заклади професійно-технічної і вищої освіти України до співпраці задля надання студентам та викладачам зручного інструменту для перегляду розкладу занять – додатка UniSched.

## Технічні вимоги

Для використання UniSched у вашому закладі професійно-технічної чи вищої освіти має функціонувати система «Деканат» авторства [ПП «Політек-СОФТ»](http://www.politek-soft.kiev.ua/) з програмним модулем «ПС-Розклад», у налаштуваннях якого дозволено доступ до API, а також доступ до посилань на онлайнові заняття через API – на той випадок, якщо у вашому закладі освіти проводяться заняття у онлайн-форматі.

!!! note "Технічна підтримка"

    За необхідності, ми безкоштовно допоможемо вашому закладу освіти здійснити відповідне налаштування програмного забезпечення [ПП «Політек-СОФТ»](http://www.politek-soft.kiev.ua/) для його повної сумісності з додатком UniSched.

## Умови співпраці

- UniSched є некомерційним проєктом, метою якого є розвток екосистеми онлайн-сервісів, які доступні для українського студентства і освітянської спільноти.
- Ви маєте визначити контактну особу, яка буде відповідати за комунікацію з командою проєкту, матиме повний адміністративний доступ до системи «Деканат», подаватиме дані щодо зборів на ЗСУ, інформацію про які потірбно поширювати у застосунку.
- Ви маєте поширювати промо-матеріали щодо можливості використання UniSched серед ваших студентів та викладачів (за такої необхідності), у тому числі на офіційних сайтах закладу освіти, його структурних підрозділів, соціальних мережах тощо.
- Здійснення платежу на перемогу України будь-якій організації, що займається волонтерською діяльністю і постачає необхідні речі на потреби Сил Оборони України, або наявність загальноуніверситетських проєктів, спрямованих на підтримку Сил Оборони і зміцнення обороноздатності України (за останні 3 місяці).
